First of all, make sure that you registered successfully into AuthX, and you have received an API KEY. Now use the following APIs in order to authorize your clients.

Add Client

You can add a client with the credentials that you want. Just put the client credentials inside a JSON object and send it to server. You will get an ID in return. This ID is the client ID which will be needed for user Login. 

curl --location --request PUT 'http://127.0.0.1:8080/api/app/{API_KEY}' \
  --header 'Content-Type: application/json' \
  --data '{
      "email": "amirhossein",
      "password": "1234"
  }'
Your input data can be any credential that you want to store for each client. It should be in JSON format. 
                  
                      {
                        "email": "something@gmail.com",
                        "username": "someone"
                      }
A simple response would be like this, which is the Client ID: 

aac14ec0-8
This client ID is what you send to your users to use for login.

Check Client

With this API, you can check if a client exists or not, if the client exists, you will get a token for getting that client with that token. 

curl --location --request POST 'http://127.0.0.1:8080/api/app/{API_KEY}?id=aac14ec0-0' \
  --header 'Content-Type: application/json'
Remember to put client ID as a query param in url.
A simple response would be like this, which is the Token for getting user: 

eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJV5IjoiOWRiOTQ4ODctZjYzOpZW50X2lkIjoiZGE0ZiJ9.s_fzqiRtT9cam14xKFZw5A3clHVFOWdJi7COdCHLK_8

Get Client

Get the client credentials by sending a get request with the token that you got from check API. 

curl --location 'http://127.0.0.1:8080/api/app/{API_KEY}?token=eyJhbGciOiJIUzR5cCRiOTQ4ODctZjYzOC002lkIjoiZGE0ZjFlZjgtNiJ9.s_fzqiRtT9cam14xKFZw5A3clHVFOWdJi7COdCHLK_8'
Remember to send the user token as a query param in url.
A simple response would be like this, which is the Client credentials: 

{
    "email": "amirhossein",
    "password": "1234"
}
